- ⛓️ Now building on **Solidity + Foundry** — 11+ smart contract projects covering DEX swaps, staking, crowdfunding, and NFT credentials.
- 🔐 Security-first approach: CEI patterns, reentrancy guards, and fuzz testing on every contract.
- 🖥️ Full stack Web3: **Next.js, wagmi, viem, RainbowKit** on the frontend, deployed on Ethereum Sepolia.
- 🎓 Currently at **Blockchain Accelerator**, sharpening the path from fintech engineering into DeFi.
- 🌎 Based in Monterrey, Mexico — open to remote roles, international and LATAM.
